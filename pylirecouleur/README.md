# pylirecouleur (vendored)

Paquet pylirecouleur embarqué pour Dys'Positif.

- Contient le package `lirecouleur/` avec la logique de syllabation.
- Les données (automates / dictionnaires) sont dans `lirecouleur/appdata/*.json`.
- Pour le développement : `PYLIRE_DEV=1 python -m pip install -e ./pylirecouleur`
- Pour installer proprement : `python -m pip install --upgrade ./pylirecouleur`
